- Ứng dụng quản lý nhà hàng Panda. <br> 
- Ngôn ngữ sử dụng: `Java` <br>
- Công cụ thiết kế: `Java swing`  <br>
- Cơ sở dữ liệu: `Mysql`  <br>
- IDE: `Netbeans`  <br>
- Công cụ quản lý phiên bản: `git`  <br>
- Công cụ quản lý mã nguồn: `github`  <br>
